A simple WebSocket client GUI ideal for development testing, built on top of Electron.

## Preview

![Preview](./assets/preview.png)

## Scripts

### Run in Development Mode

```
yarn dev
```

### Create a Production Build

```
yarn build
```

### Run Production Build

```
yarn start
```
