A simple WebSocket client GUI, built on top of Electron.

## Preview

![Preview](./assets/preview.png)

## Scripts

### Run in Development Mode

```
yarn dev
```

### Create a Production Build

```
yarn build
```

### Run in Production Mode

```
yarn start
```
